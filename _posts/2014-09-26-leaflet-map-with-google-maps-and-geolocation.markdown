[Here](/files/superleaflet.html) is a [leaflet.js](http://leafletjs.com) map, using plugins to show various sources including OpenStreetmap, MapBox and Google Maps.  It also includes a search facility using the OSM geolocator.

It's mostly here for my own development use.
You can pass a lat/lng on the query string to auto centre and zoom the mapp when loaded, e.g. [lat=51.5&lng=0&zoom=16](/files/superleaflet.html?lat=51.5&lng=0&zoom=16) gives us ![greenwich](/files/leaflet_greenwich.png)
